# tourbillon excludes for linux user-home backups (rsync --exclude-from=)
#
# Patterns are matched anywhere in the source tree unless they start with '/'.
# Goal: skip regenerable cruft so backups stay focused on irreplaceable data.
#
# Per-host extras can be added at configs/hosts/excludes/<hostname>-extras.txt
# and chained in the per-host config.
#
# Trash (~/.local/share/Trash/) is INCLUDED on purpose. The kodiak-side
# snapshot retention covers "wait, I shouldn't have deleted that" recovery
# for a couple weeks. See doc/ADR-002-host-backups-linux.md and the
# memory note "safety-nets-for-scratch."
